# Smart Utility Toolkit 🚀

A premium, high-performance Flutter utility application designed with glassmorphic aesthetics and advanced feature sets. Built as part of the **HNG14 Internship Stage 1 Mobile Track**.

## ✨ Features

### 📋 Task Manager (Stage 1 Core)
- **Full CRUD operations**: Create, Read, Update, and Delete tasks.
- **Persistent Local Storage**: Tasks are saved offline using `shared_preferences`.
- **Productivity Insights**: Visual completion stats using custom charts.
- **Prioritization**: Categorize and prioritize tasks (Low, Medium, High).
- **Premium UX**: Fluid swipe-to-action gestures and real-time smart search.

### 🛠 Utility Tools (Stage 0 Baseline)
- **Currency Converter**: Dynamic conversion with clipboard intelligence.
- **BMI Calculator**: Health metrics with visual feedback.
- **Fuel Efficiency**: Trip cost and consumption calculator.
- **Unit Converters**: High-precision conversions for Length, Weight, and Temperature.

### 🎨 Design & Experience
- **Glassmorphism UI**: Premium frosted glass effects throughout the app.
- **Command Palette**: Global search to jump to any tool or action instantly.
- **Dynamic Theming**: Adaptive Light/Dark mode toggled via a centralized control.
- **Haptic Intelligence**: Tactile feedback for every significant interaction.
- **Fluid Animations**: Powered by `flutter_animate` for a polished feel.

## 🛠 Tech Stack
- **Framework**: [Flutter](https://flutter.dev) (Dart)
- **State Management**: [Riverpod](https://riverpod.dev)
- **Persistence**: `shared_preferences`
- **Icons**: `lucide_icons`
- **Animations**: `flutter_animate`
- **Charts**: `fl_chart`
- **UUID**: `uuid` for unique identifiers

## 🚀 Getting Started

### Prerequisites
- Flutter SDK (Latest Stable)
- Git

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/uchekingsley/Toolkit-App.git
   ```
2. Navigate to project directory:
   ```bash
   cd smart_utility_toolkit
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```
