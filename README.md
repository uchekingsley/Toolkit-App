# Smart Utility Toolkit

A premium, intelligent utility suite built with Flutter. This toolkit provides a range of essential tools including unit converters, financial calculators, health trackers, and a comprehensive Task Manager, all wrapped in a sleek, glassmorphic UI.

## Features

- **Unit Converters**: Length, Weight, Temperature, and Currency.
- **Calculators**: BMI, Fuel, and Tip calculators with history persistence.
- **Task Manager**: A full-featured checklist manager with categories, priorities, and local storage.
- **Global Search**: Command palette for quick access to any tool.
- **Premium Design**: Modern glassmorphic aesthetics with dynamic parallax motion.
